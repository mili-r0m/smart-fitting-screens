# 👚 Smart Fitting Screens
> Sistema inteligente de recomendaciones para probadores de ropa utilizando Inteligencia Artificial.

---

## 📌 Contexto del Proyecto

**Smart Fitting Screens** es un sistema de recomendaciones con Inteligencia Artificial diseñado para locales de indumentaria. La solución consiste en una pantalla empotrada dentro del probador que interactúa con el cliente durante su experiencia de compra.

Al ingresar al probador, el usuario escanea la prenda que desea probar. A partir de esa información, el sistema muestra:

- Descuentos y promociones disponibles.
- Talles y colores disponibles.
- Recomendaciones personalizadas generadas mediante IA de prendas complementarias o similares.

---

### 🎯 Objetivos

- Mejorar la experiencia de compra dentro del probador.
- Fomentar la autonomía del cliente durante la elección de prendas.
- Incrementar las ventas mediante recomendaciones inteligentes (cross-selling).
- Brindar una plataforma flexible para integrar promociones y campañas comerciales.
